# Spam Message Detection using Machine Learning

This project builds a spam message detection system using machine learning.

The model classifies SMS messages as **Spam** or **Ham (Not Spam)**.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Tkinter

## Machine Learning Model
Multinomial Naive Bayes

## Features
- Text preprocessing
- Bag of Words vectorization
- Spam classification model
- Desktop GUI application using Tkinter

## How to Run

1. Clone the repository

git clone https://github.com/niranjan0342/spam-email-detector.git

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

## Example

Input:
Free iPhone offer click now

Output:
Spam Message
